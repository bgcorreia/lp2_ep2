# Project - LP2 - EP2

To compile/linker execute `make` in the clone directory.

Example:

```bash
user@host:~/lp2_ep2$ make
g++ BanheiroUnissex.cpp -o BanheiroUnissex -std=c++11 -lpthread
user@host:~/lp2_ep2$
```
Then, execute the generated binary:

```bash
user@host:~/lp2_ep2$ ./BanheiroUnissex
OUTPUT...
```
